# TrafficEmail

A traffic congestion detection and email alert system built using Python, Streamlit, OpenCV, and YOLOv5.

## Features

- Detects traffic congestion from video feeds
- Uses YOLOv5 for vehicle detection
- Displays traffic statistics
- Sends email alerts automatically
- Interactive Streamlit dashboard

## Technologies Used

- Python
- Streamlit
- OpenCV
- YOLOv5
- Pandas

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
